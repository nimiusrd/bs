# セクション構造のサンプル {#sections}

## はじめに {#intro}

このページは見出し階層と目次出力が意図どおりになることを確認するためのフィクスチャです。

## 使い方 {#usage}

### インストール {#install}

```
npm install
```

### コマンド {#commands}

- `npm run build:docs`: 本番コンテンツをdistへ変換
- `npm run test:fixtures`: フィクスチャの期待出力と比較

## 注意事項 {#notes}

- 見出しにIDを付けることで、アンカーの生成と目次が安定します。
